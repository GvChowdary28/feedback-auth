Steps to use feedback application

  1. npm install -g @angular/cli [If cli is not installed else skip].
  2. On Project path use command npm install to install all the dependencies.
  3. On failure to install dependencies use npm install --force so it can take compatiable dependencies.
  4. Run ng serve --open for running project on localhost:4200.
  5. To run Json server use command npm run json-server [important to perform http calls].
